HTML stands for Hypertext Markup Language and is used to create webpages

CSS stands for Cascading Style Sheets and is used to design webpages and make them responsive

JavaScript is a programming language used to make web pages dynamic and interactive

Vamos um pouco mais fundo no assunto, aprender sobre um tipo especial de tag e explorar alguns atributos HTML que podem ser usados para modificar elementos HTML. A razão de termos que abrir e fechar uma tag é simplesmente para saber onde o elemento começa e onde ela termina.

Porém, existem elementos HTML que não possui nenhum conteudo, portanto, eles não precisam de uma tag para fecha-lo. Esses elementos são chamados de self-closing-tag, e a primeira que você deveria conhecer é a tag br

BR pula uma linha na sua webpage. Embora nao seja necessário, é tambem uma boa prática, adicionar uma / após a tag de fechamento automático, e embora não pareça importante faze-lo apenas por boas práticas, esses pequenos detalhes, mostrarão que você ja sabe se portar como um profissional, além de já ir praticando para quando for usar frameworks, como o react por exemplo, que EXIGE que você coloque a barra após usar uma self-closing-tag.

HR é uma self-closing-tag que coloca uma linha horizontalmente no seu texto, podendo separar assim o conteúdo do seu site. É muito indicado para separar o início do conteúdo do seu website.

INPUT é uma self-closing-tag muito interessante e complexa, pois cria um campo para que o usuário possa colocar um texto. Mais adiante no curso, isso nos irá permitir a coletar inumeras informações sobre o usuário, pois existem vários tipos de input, e isso pode ir de tipos simples como textos, números ou check-boxes para alguns mais complexos, como dados, seletor de horário, seletor de cor ou até mesmo um upload de um arquivo, mas por enquanto vamos aprender como modificar esse elemento input.

Atributos HTML

Com atributos em html, você pode modificar ou colocar informação adicional aos elementos. Funciona da seguinte maneira: Um atributo é escrito dentro de uma tag aberta em HTML.

Você coloca o cursor após o nome do elemento e insere o atributo. Neste caso o atributo usado foi o type na tag imput/, classificando-a como um campo para colocar números. Imediatamente ao salvar o documento, teremos apenas um campo para colocar números.

Também podemos colocar um placeholder em frente a classe, que irá colocar uma dica no campo criado pela tag input, portanto, colocamos uma dica que corresponda ao conteúdo da tag input.
